# Mohammad & Ammar Housing — Website Preview Demo

Interactive client presentation of the Amman Housing website screenshots.

Open [`index.html`](index.html) in any browser — no build step or server required.

## Features

- **Device frames** — browse the site inside a desktop browser mockup (1440px), an iPhone mockup (390px), or both side by side
- **Language toggle** — Arabic (RTL) and English versions of every page
- **5 pages** — Home, About, Projects, Project Detail, Contact
- **Auto-scroll tour** — hands-free scroll through each page while presenting
- **Keyboard shortcuts** — `←`/`→` switch pages, `D` cycles device, `L` toggles language, `Space` starts/stops the tour
